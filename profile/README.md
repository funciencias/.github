# FUNCIENCIAS®

**IA con resultados reales y criterio humano.**

FUNCIENCIAS es una fundación colombiana activa desde 2009, especializada en IA aplicada. Instalamos capacidades de inteligencia artificial en equipos humanos para mejorar resultados reales en licitaciones, finanzas y procesos.
